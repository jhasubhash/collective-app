# Collective

AI-Powered Collaborative Knowledge Management

**[Download](https://github.com/jhasubhash/collective-app/releases) | [Website](https://jhasubhash.github.io/collective-app)**

## Features

- **AI-Powered Chat** - Ask questions about your documents using local LLMs
- **P2P Collaboration** - Share spaces with team members, no server required
- **Offline-First** - Works without internet, syncs when connected
- **Privacy-Focused** - Your data stays on your devices

## Requirements

- **For AI features**: Install [Ollama](https://ollama.ai) with a model like `llama3.2`
- **macOS**: macOS 11 (Big Sur) or later
- **Windows**: Windows 10/11 with WebView2
- **Linux**: Ubuntu 20.04+ with WebKitGTK

## Tech Stack

- Rust + Tauri (backend)
- React + TypeScript (frontend)
- SQLite (local storage)
- libp2p (peer-to-peer networking)
